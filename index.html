Programacion web


El propósito de este sitio web es crear programas usando JavaScript para darle la funcionalidad adecuada a cada uno de los elementos que conforma HTML, por lo que este este es el propósito principal sin embargo hay mas ya que se empezó a usar estructuras condicionales y de iteración las cuales nos ayudaron demasiado para facilitar este proceso y hacerlo más fácil.

En este sitio web se encuentran dos programas los cuales uno es un ticket y otro es un examen por lo que el primero simula un ticket de compra el cual se da en todas las tiendas de compra.

El examen por otra parte es un cuestionario de 10 preguntas la cuales son en ingles para esto debes leer con atención y contestar correctamente cada uno de los incisos al final de este te mostrará tu resultado y te dirá si aprobaste o reprobaste al igual que te podrá dar como resultado si obtuviste una beca.
